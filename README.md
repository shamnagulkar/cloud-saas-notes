# Memoir

Memoir is a clean, cloud-style notes application built to capture thoughts, ideas, and small moments worth remembering.

It focuses on simplicity, usability, and the core idea behind SaaS — access your data anytime through a browser.


## What it does

- Write and save notes instantly  
- Stores notes persistently in a database  
- Shows when each note was created  
- Delete notes when they’re no longer needed  

Simple, fast, and distraction-free.


## Tech used 

- Python  
- Flask  
- SQLite  
- HTML & CSS
